#Summer Training Project Name : "Trend Trail"

Based on Example - "Levi's Exclusive Store"

Aim : Levi's Exclusive Store is based on the SaaS Model which is Software as a Service Model.
      It helps various Levi's Outlet Managers to perform CRUD Operations on the Customers and the sales data.

Database : MongoDB
           It contains an database for (1)- Users i.e Managers
                                       (2)- Customers
                                       (3)- Sales

Main Key Performance : 
1. Each Manager and their Outlet Customers data is Autheticated and Secured by Managers's Unique Email and Password set 
2. Multiple Managers can Register and add different Customers and their Sales Data
